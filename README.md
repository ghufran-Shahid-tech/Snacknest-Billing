# Snacknest Kitchen POS

A **responsive, browser-based POS system** for small restaurants and kitchens, built with **HTML, CSS, and JavaScript**. Designed for quick order entry, printing, and tracking order history.

## Features

- **Menu Management**
  - Categorized menu: Rice & Pasta, Burgers, Snacks, Pizzas, Drinks
  - Add custom items with price
  - Quantity management with plus/minus buttons

- **Bill Management**
  - Real-time subtotal, delivery, and total calculation
  - Delivery charges selection (preset or custom)
  - Token number management
  - Print-ready layout
  - Share bill via WhatsApp

- **History & Gallery**
  - Last 10 orders saved in local storage
  - Gallery section to showcase images or menu items
  - Switch between History and Gallery using buttons

- **Responsive Design**
  - Works on desktop, tablet, and mobile
  - Smooth animations for page and menu loading

- **Dark Mode**
  - Toggle between light and dark themes

## How to Use

1. Clone or download the repository.
2. Open `index.html` in a browser.
3. Select a category and add items to the bill.
4. Enter customer details and select order type and payment method.
5. Set delivery charges or add custom delivery.
6. Print the bill or share via WhatsApp.
7. Switch between History and Gallery sections at the bottom.

## Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
- LocalStorage for saving history

## Customization

- Update `menuData` object in the script to change menu items.
- Replace `logo.png` with your own logo.
- Add/remove images in the gallery section.
